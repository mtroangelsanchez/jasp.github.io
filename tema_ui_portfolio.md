# Tema UI Portfolio - Estructura y Diseño

## Fuente Original
- URL: https://wp.w3layouts.com/ui-portfolio/
- Tipo: Portfolio personal / One-page
- Estilo: Moderno, limpio, minimalista con tonos oscuros y acentos verdes

---

## Estructura de Secciones

### 1. Hero / Banner
- Título grande: "Hola, soy [Nombre]"
- Subtítulo: Rol profesional
- Descripción breve
- Botones CTA: "Contáctame" / "Mi Portfolio"
- Imagen de banner lateral
- Fondo: Gradiente oscuro verde-azulado

### 2. Biografía (About)
- Encabezado: "MI BIOGRAFÍA"
- Título: Rol + ubicación
- Descripción personal
- Datos personales: Nombre, Ubicación, Email, Idiomas
- Redes sociales (LinkedIn, GitHub, Twitter)
- Botones: "Contáctame" / "Descargar CV"

### 3. Servicios (What I Do)
- Título: "Lo Que Puedo Hacer Por Ti"
- Grid de 6 cards con íconos circulares gradiente:
  - Arquitectura de Software
  - Desarrollo Full Stack
  - Cloud & DevOps
  - Inteligencia Artificial
  - Desarrollo Móvil
  - Seguridad & Pentesting

### 4. Skills
- Encabezado: "MIS HABILIDADES"
- Título: "Habilidades Destacadas"
- Barras de progreso con porcentajes y gradiente verde
- 8 habilidades principales

### 5. Estadísticas / Counters
- Fondo oscuro verde-azulado
- 4 contadores: Años experiencia, Proyectos, Grados académicos, Certificaciones

### 6. Portfolio / Trabajos
- Título: "Proyectos Destacados"
- Grid 3x2 de imágenes con overlay verde al hover
- Categoría + descripción corta

### 7. Experiencia Profesional
- Timeline vertical con línea gradiente verde
- Dots circulares verdes
- Cards con fecha, puesto, empresa y descripción

### 8. Educación
- Grid de 3 cards con íconos de graduación
- Universidad, grado y año

### 9. Logos de Tecnologías
- Fila de logos en escala de grises
- Color al hover
- SVG inline para independencia de archivos externos

### 10. Contacto
- Grid 2 columnas: Info + Formulario
- Íconos circulares con fondo verde claro
- Formulario con inputs redondeados

### 11. Footer
- Fondo oscuro verde-azulado
- Logo/Nombre del sitio
- Links de navegación
- Redes sociales
- Copyright

---

## Paleta de Colores
- Primario: #0d9e71 (verde esmeralda/teal)
- Primario oscuro: #0a7d5a
- Secundario: #1a1a1a (negro suave)
- Texto: #333333
- Texto claro: #666666
- Fondo blanco: #ffffff
- Fondo claro: #f8f9fa
- Fondo oscuro: #1c2e3a (verde-azulado oscuro)
- Acento gradiente: #0d9e71 → #34d399
- Hero gradiente: #1c2e3a → #2a4a5a → #1c3a3a
- Stats/Footer: #1c2e3a → #2a4a5a

## Tipografía
- Fuente: Poppins (Google Fonts)
- Pesos: 300, 400, 500, 600, 700
- Títulos: 700 (bold)
- Subtítulos: 600 (semi-bold)
- Cuerpo: 400 (regular)
- Tags: 600 con letter-spacing 2px

## Componentes de Diseño
- Border radius: 12px (cards), 50px (botones)
- Sombras: 0 10px 30px rgba(0,0,0,0.1)
- Sombra hover: 0 20px 40px rgba(13,158,113,0.2)
- Transiciones: all 0.3s ease
- Animación hero: float 3s ease-in-out infinite

## Características
- Diseño responsive (breakpoints: 992px, 768px, 480px)
- One-page con scroll suave
- Navbar transparente → blanca al scroll
- Animaciones de reveal al scroll
- Barras de skill animadas
- Compatible con Google Sites (HTML/CSS estático)
- Íconos: Font Awesome 6.4.0 (CDN)
- Logos de tecnologías en SVG inline
